# Job Application Form

## Description

This is a beginner-level web project that I created during my first year of college as part of my frontend development practice. The project is a Job Application Form built using HTML, CSS, and JavaScript. It allows users to enter their personal details, select a job role, and display submitted information in a table format.

This project helped me improve my understanding of form creation, page styling, DOM manipulation, and basic JavaScript functions.

## Features

- Responsive Form Layout
- Clean and Interactive UI
- Dark Mode Toggle
- Live Date and Digital Clock
- Job Role Dropdown Selection
- Submit Data and Show in Table
- Reset Form Fields
- Close/Hide Submitted Table
- Hover Effects and Animations

## Built With

- HTML5
- CSS3
- JavaScript
